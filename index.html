Core Technology
온톨로지란 무엇인가?
온톨로지는 MES·SCM·ERP 도메인의 업무 개념, 관계, 규칙을 구조화한 지식 체계입니다.
AI가 "생산실적"이 어떤 데이터와 연결되는지, 어떤 UI 컴포넌트가 필요한지를 판단하는 고정된 도메인 기준점으로 활용됩니다.
1
도메인 업무 개념 구조화 같은 용어도 업무 맥락에 따라 의미를 구분하여 AI Hallucination 방지
2
모호성 Q&A 생성 기준 누락·모호한 요건을 분석해 AI가 정확한 추가 질문을 생성
3
화면 생성 품질 일관성 유지 도메인별 Layout·컴포넌트·Workflow를 일관되게 생성
4
유사 설계 탐색 및 재사용 GraphRAG 기반으로 기존 설계 자산과 유사도 비교·재활용

Impact
화면설계, 이렇게 달라집니다
화면 초안 작성부터 검토 승인, 화면설계서 생성까지 실제 결과물 기준으로 비교했습니다.
Effect 01
화면 초안 작성
4시간10분 이내
초안 작성에 하루를 쓰던 흐름에서, 대화형 생성으로 바로 검토 가능한 첫 화면을 확보합니다.
Effect 02
검토 및 승인 왕복 시간
50% 단축
검토 요청, 피드백, 승인 이력을 한 흐름으로 묶어 커뮤니케이션 왕복 비용을 줄입니다.
Effect 03
화면설계서 파일 작성시간
2분
목업과 스펙을 바탕으로 설계서 산출물까지 자동 생성해 별도 문서 작성 단계를 없앱니다.
Effect 04
Vision Embedding(CLIP) 유사도
90%
기존 설계 결과물과 높은 수준으로 유사한 초안을 빠르게 생성해 수정 부담을 낮춥니다.
